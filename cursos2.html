Diseñador web

Da tus primeros pasos en mundo del desarrollo web, conociendo a full HTML y CSS. Aprende lenguaje y buenas prácticas. Descubre el poder de Git y crea tu perfil en Github.

Ver programa completo

Acerca del curso

Conviértete en diseñador web con nuestro curso de Fundamentos web y nuestro moderno plan de estudio que incluye las últimas tecnologías: HTML, CSS, SASS, GIT, Github, Bootstrap, Accesibilidad, SEO, Hosting y servidores web.

Semana 1: Git y HTML.

  • Aprende a utilizar Git
  • Crea tu perfil en Github
  • Utiliza correctamente las etiquetas de HTML
  • Conoce las etiquetas semánticas de HTML5

Semana 2: CSS.

  • Conoce todos secretos de CSS
  • Aprende buenas prácticas de estilado
