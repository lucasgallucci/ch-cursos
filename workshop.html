Información general

Introducción a React y tecnologías afines. Introducción a Webpack (un bundler que te permitirá trabajar con React) y sus principales ventajas. Aprenderás a organizar una aplicación y a crear sus diferentes componentes. Comprensión de la sintaxis de JSX y cómo interactúa con el DOM. Vas a construir una aplicación básica consultando con una API, que muestre la información en la app. Vas a ver un poco de flux y las diferentes alternativas de implementación.

Qué vas a aprender

  • React
  • Webpack
  • Arquitectura de una App React

Pre-requisitos

  • JavaScript Intermedio o Avanzado
  • Conocimiento sobre algún bundler (tipo Gulp o similar)
  • HTML & CSS básico
